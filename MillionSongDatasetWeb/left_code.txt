<h2>General / Requirements</h2><p>
All our code is available on <a href="https://github.com/tb2332/MSongsDB">github</a>
(you don't need git installed).<p>
Files are in HDF5 format. Why? See FAQ.
Except for new MATLABs (2009b), you need the HDF5 library to use the dataset 
<a href="http://www.hdfgroup.org/downloads/">here</a>.
Other requirements depend on the language you use, see the following
section. Note that in Ubuntu, install everything (including optional packages) with:<p>
<small>
<div style="border:1px solid #808080;padding:1.2em; background-color:#f0f0f0; margin-left: 25px; margin-right: 0px;">
<div style="margin-left: 25px;">
<pre>
sudo apt-get install libhdf5-serial-1.8.4 libhdf5-serial-dev 
    python-tables python-tables-doc libjhdf5-java h5utils hdfview
</pre>
</div>
</div>
</small>
<p>
We believe that more than 90% of the researchers using the dataset will not
need to understand the structure of the HDF5 files. Therefore, the wrappers
below simply provide getters to the different fields saved in each file.
If you really want to speed things up and want to play with HDF5, we
have some code to aggregate many files into one (thus saving on IO operations!).
Want to dig deeper? All our code is open source, the HDF5 file descriptors
are available in python, see the latter section on Dataset Creation.

</div>

<h2>Wrappers</h2><p>
We provide wrappers for <a href='#python'>python</a>, <a href='#matlab'>matlab</a>
and <a href='#java'>java</a>, plus a note on <a href='#cplusplus'>C/C++</a>.<p>

<a name='python'><h3>Python</h3></a>
The HDF5 wrapper in python is <a href="http://www.pytables.org/moin">pytables</a>.
It relies on <a href="http://numpy.scipy.org/">numpy</a> (that you probably already have).
If you just want to use the HDF5 provided in the dataset, the file
<a href="https://github.com/tb2332/MSongsDB/blob/master/PythonSrc/hdf5_getters.py">
hdf5_getters.py</a> should be enough:<p>
<small>
<div style="border:1px solid #808080;padding:1.2em; background-color:#f0f0f0; margin-left: 25px; margin-right: 140px;">
<div style="margin-left: 25px;">
<pre>
import hdf5_utils
import hdf5_getters
h5 = hdf5_utils.open_h5_file_read(<path some file>)
duration = hdf5_hetters.get_duration(h5)
h5.close()
</pre>
</div>
</div>
</small><p>
If you want a deeper understanding of the
dataset, or want to combine many files into one, see the section on
<a href='#datacreation'>dataset creation</a> below. All our work was done in python.<p>

<a name='java'><h3>Java</h3></a>
You need some JAR files accessible <a href="http://www.hdfgroup.org/hdf-java-html/">here</a>. 
Read the <a href="https://github.com/tb2332/MSongsDB/tree/master/JavaSrc/">README</a> file, 
but the class hdf5_getters.java should give
you all you need.<p>

<a name='cplusplus'><h3>C/C++</h3></a>
We do not have these wrappers for the moment. That being said, HDF5 is
a C library and we have some testing code in C++, it is definitely feasible.
If you need these for your application, let us know, we'll work on it!<p>
